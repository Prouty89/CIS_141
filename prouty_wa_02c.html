CIS 132 - Intro to Computer Programming

Online

Course Objective:

This course is an introductory course to introduce the student to the basic concepts and techniques used in programming. Writing, debugging, and testing business applications. This course will cover the creation, compilation, and execution of C# programs using Visual Studio. Students will write and understand C# language constructs, syntax, logic, and semantics. Students will develop fundamental programming and trouble-shooting skills.
